### 二、运行环境及运行方法

#### 2.1 测试运行环境

操作系统：windows10，11

python版本：Python 3.7以上 （其余版本可能会在安装mediapipe时出错）

#### 2.2 运行方法

`$ git clone https://github.com/enpeizhao/CVprojects.git`

`$ cd CVprojects`

`$ pip3 install -r requirements.txt`

`$ cd codes`

`$ python 6.hand_control_volume/6.hand_control_volume.py`
